Airline Reservation System

A modern airline reservation system project. This system includes core airline automation features such as flight search, reservation management, seat selection, payment processing, passenger management, and an admin dashboard.

Project Overview

This project is designed to simulate a real-world airline reservation system using object-oriented programming principles and modern web technologies.

The system allows users to:

Search for flights
Make reservations
Select seats
Manage passenger information
Process payments
Generate tickets

Administrators can manage flights, reservations, and system data through a dedicated admin dashboard.

Havayolu Rezervasyon Sistemi
Bu proje, kullanıcıların uçuş arayabildiği, rezervasyon oluşturabildiği ve koltuk seçimi yapabildiği modern bir havayolu rezervasyon sistemi uygulamasıdır. 
Sistem içerisinde yolcu yönetimi, ödeme işlemleri, bilet oluşturma ve admin paneli gibi temel havayolu otomasyon özellikleri bulunmaktadır.
Proje Java tabanlı backend yapısı ile geliştirilmiş olup frontend tarafında HTML, CSS ve JavaScript kullanılmıştır.
Eğitim ve akademik amaçlarla geliştirilen bu sistem, nesne yönelimli programlama mantığını ve gerçek dünya rezervasyon süreçlerini simüle etmeyi amaçlamaktadır.
